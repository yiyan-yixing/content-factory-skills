# 架构决策记录

> 此文件由所有 Agent 共享。每次重大架构/技术选型决策时更新。

## 当前架构

- **阶段**：0→1 创业期
- **模式**：AI 辅助内容生产（人创意 + AI 执行 + 数据验证）
- **核心约束**：8 层智能体闭环，L3 生产层优先 Agent 化

## 内容工厂八层架构

```
L0 战略层    ──→  L1 设定层    ──→  L2 剧本层    ──→  L3 生产层
用户画像/选题      世界观/人物       故事/爆点/节奏     场景/对话/章节
@主编              @设定师           @编剧              @写手

L4 审稿层    ──→  L5 运营层    ──→  L6 商务层    ──→  L7 复盘层
结构/情绪/一致性    转化/社群/传播    会员/付费/IP运营    反馈/复盘/沉淀
@审稿              @运营             @商务              @复盘官

                                                    ↓ 反馈闭环
                                              L0 主编 ← L7 复盘官
```

- **L0-L2（战略→设定→剧本）**：创意主导，AI 辅助，人做关键决策
- **L3（生产层）**：Agent 优先，规则明确，输入输出清晰
- **L4（审稿层）**：AI 审校 + 人工抽检，一致性是底线
- **L5-L6（运营→商务）**：数据驱动，持续优化
- **L7（复盘层）**：闭环守护，经验沉淀驱动下一轮

## ADR（Architecture Decision Records）

格式：`ADR-<编号> | <标题> | <状态>`

### ADR 模板

```
## ADR-<N>: <标题>

### 状态
提议 / 已采纳 / 已废弃 / 已替代

### 背景
为什么要做这个决策？

### 决策
我们选择了什么？

### 理由
为什么这么选？考虑了哪些替代方案？

### 后果
这个决策带来的好处和代价？
```
