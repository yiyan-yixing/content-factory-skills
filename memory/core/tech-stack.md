## 内容工厂技术栈分层

| 层级 | 说明 | 典型选型 |
|------|------|----------|
| 创作平台 | 连载发布 | 起点/番茄/晋江/知乎 |
| 写作工具 | 章节编辑/版本管理 | Markdown + Git / 在线编辑器 |
| AI 辅助 | 生成/审校/对齐 | Claude / GPT / 自训微调 |
| 数据采集 | 读者行为/评论/阅读统计 | 平台后台 + 爬虫 + API |
| 数据分析 | 完读率/跳出点/付费转化 | Python + SQL + BI 工具 |
| 社群运营 | 读者社群/互动 | 微信群/QQ群/贴吧/Discord |
| 商业化 | 付费/会员/衍生 | 平台分成 + 自建站 + Stripe |

## 选型原则

- **平台优先**：先在成熟平台验证，不自建站
- **AI 赋能不替代**：AI 辅助创作和审校，核心创意由人把控
- **数据驱动**：用完读率/付费转化数据说话，不靠直觉
- **速度优先**：从选题到首章发布 < 1 周
- **可复用**：世界观/角色/风格资产跨作品复用
