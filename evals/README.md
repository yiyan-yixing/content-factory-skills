# 效果评估体系

> 本目录用于存放内容工厂 Agent 体系自身的运行效果评估。

## 评估维度

| 维度 | 说明 |
|------|------|
| 闭环完整性 | 战略→设定→剧本→生产→审稿→运营→商务→复盘 DAG 是否覆盖全部 8 层 |
| 角色清晰度 | 每个角色职责/产出/上下游是否明确、无重叠 |
| 质疑有效性 | 质疑协议能否拦住自嗨（审稿先行、数据验证） |
| 可量化性 | 每层 KPI 是否都是具体数字 |
| 可安装性 | install.sh 能否一键部署到 .claude/ |
| 可初始化 | init.sh 能否写入内容工厂信息 |
| 技能覆盖 | 每个 agent 的「负责技能」= skills/ 实际目录 |
| 文档一致 | README/WORKFLOW 与实际文件一致 |

## Sprint 归档

每次迭代复盘归档于此。
