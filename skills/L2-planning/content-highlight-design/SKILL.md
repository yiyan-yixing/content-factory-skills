# SKILL-203：爆点设计

你是内容公司的爆点策划。你的目标：在每章植入至少一个让读者"忍不住转发"的高光时刻。

## 技能定义

| 维度 | 说明 |
|------|------|
| **输入** | 章节计划、情绪地图 |
| **输出** | HighlightSet（爆点集） |
| **依赖** | SKILL-202 长线规划、SKILL-002 情绪洞察 |
| **自动化** | 人工★★★☆☆ Agent★★★☆☆ |
| **训练价值** | 高 |

## 执行步骤

### Step 1：爆点类型选择（10min）

六大爆点类型：
| 类型 | 效果 | 示例 |
|------|------|------|
| 金句 | 截图传播 | "我不需要被喜欢，我需要被尊重" |
| 反转 | 讨论炸裂 | 闺蜜竟然是幕后黑手 |
| 泪点 | 情感共鸣 | 他终于说了那句话 |
| 爽点 | 释放快感 | 反击成功 |
| CP | 磕糖传播 | 暗戳戳的双向奔赴 |
| 悬念 | 催更动力 | 门外站的人是谁 |

### Step 2：植入位置（10min）

每章的爆点分布：
- 章 1/3 处：小钩子（保持注意力）
- 章 2/3 处：中爆点（情绪升温）
- 章末：大钩子（必须看下一章）

### Step 3：情绪映射（10min）

爆点必须匹配目标情绪：
- EmotionMap 的 Top 痛点 → 泪点/共鸣
- EmotionMap 的 Top 幻想 → 爽点/CP
- EmotionMap 的 Top 恐惧 → 悬念/反转

## 产出

1. HighlightSet（每章爆点清单）
2. 爆点类型分布图
3. 传播潜力评估

## 关键指标

- 传播率：分享人数 ÷ 活跃人数，目标 > 20%
- 讨论度：爆点出现后的评论增长率

## 反模式（避免）

- ❌ 每章全是爆点（读者疲劳）
- ❌ 爆点与情绪脱节（炫技无共鸣）
- ❌ 只有一种类型（全爽或全虐 = 单调）
- ❌ 爆点靠巧合（读者会觉得假）

## 资产沉淀

- HighlightSet → 下游技能（场景设计、正文生成）的写作指引
- 爆点模板库是内容公司的核心资产
