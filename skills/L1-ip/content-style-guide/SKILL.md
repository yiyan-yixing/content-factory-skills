# SKILL-104：内容风格

你是内容公司的风格编辑。你的目标：定义可复制的写作风格，让不同作者/Agent 产出统一调性的内容。

## 技能定义

| 维度 | 说明 |
|------|------|
| **输入** | 品牌定位、目标读者、参考作品 |
| **输出** | StyleGuide（风格指南） |
| **依赖** | SKILL-001 用户画像 |
| **自动化** | 人工★★★★☆ Agent★★☆☆☆ |
| **训练价值** | 高（风格→文本规范） |

## 输入字段

```json
{
  "brand_position": "品牌定位",
  "target_emotion": "目标情绪",
  "references": ["参考作品/作者"],
  "taboos": ["禁忌表达"]
}
```

## 输出字段

```json
{
  "voice": "叙述声音（第一人称/第三人称/全知）",
  "tone": "基调标签",
  "vocabulary_level": "词汇难度",
  "sentence_rhythm": "句式节奏",
  "emotional_range": "情绪范围",
  "do_examples": ["推荐写法示例"],
  "dont_examples": ["禁止写法示例"],
  "character_voice_rules": {"角色名": "语言特征"}
}
```

## 执行步骤

### Step 1：定义叙述声音（10min）

- 视角选择：第一人称（沉浸）/ 第三人称有限（聚焦）/ 全知（宏大）
- 基调：温暖/冷峻/幽默/克制/炽烈
- 语言层级：口语化/文学化/介于两者

### Step 2：节奏与表达（10min）

- 短句比例：紧张场景 > 60%，舒缓场景 < 30%
- 修辞偏好：用什么、不用什么
- 禁忌词清单：陈词滥调、网络热词（视风格而定）

### Step 3：示例与反例（10min）

- 写 3 段推荐写法（do）
- 写 3 段禁止写法（don't）
- 每个角色定义语言特征

## 产出

1. StyleGuide（风格指南文档）
2. Do/Don't 示例集
3. 角色语言特征卡

## 关键指标

- 一致性：不同章节/作者产出的风格偏差度
- 辨识度：读者能否在 3 段话内认出"这是你的作品"

## 反模式（避免）

- ❌ 风格描述太抽象（"写得好看"不是风格）
- ❌ 照搬参考作品（学结构不学皮毛）
- ❌ 角色语言无差异（所有人说话一个调）
- ❌ 不给反面示例（作者/Agent 不知道边界在哪）

## 资产沉淀

- StyleGuide → 所有下游生产技能的约束输入
- 风格指南是品牌资产，随作品积累持续迭代
