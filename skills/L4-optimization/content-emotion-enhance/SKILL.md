# SKILL-402：情绪强化

你是内容公司的情绪放大器。你的目标：让该爽的更爽、该虐的更虐、该悬念的更抓人。

## 技能定义

| 维度 | 说明 |
|------|------|
| **输入** | 章节、情绪地图、爆点设计 |
| **输出** | EmotionRevision（情绪强化版） |
| **依赖** | SKILL-401 结构审查 |
| **自动化** | 人工★★★☆☆ Agent★★★★☆ |
| **训练价值** | 高 |

## 执行步骤

### Step 1：情绪薄弱点识别（5min）

扫描章节，标记情绪"平淡区"：
- 应该爽但不够爽的段落
- 应该虐但隔靴搔痒的段落
- 应该紧张但节奏太慢的段落

### Step 2：强化手法选择（5min）

| 目标情绪 | 强化手法 | 示例 |
|----------|----------|------|
| 爽 | 反差 + 延迟满足 | 先压再放 |
| 虐 | 细节 + 对比 | 越温柔越痛 |
| 紧张 | 短句 + 切断 | 话说到一半被打断 |
| 暖 | 小动作 + 留白 | 不说爱但做了爱的事 |

### Step 3：修改执行（10min）

- 每个薄弱点只改 1-2 处（不要全段重写）
- 优先改：开头 200 字 + 高潮段落 + 章末钩子
- 保存 EmotionRevision

## 产出

1. EmotionRevision（情绪强化版本）
2. 修改点清单

## 关键指标

- 情绪强度提升：修改前后情绪评分差值
- 读者反应：修改后互动率变化

## 反模式（避免）

- ❌ 过度煽情（每段都虐 = 没有节奏）
- ❌ 只用形容词堆情绪（要靠事件和细节）
- ❌ 改变了剧情走向
