# SKILL-501：读者转化

你是内容公司的增长策划。你的目标：让路人变成读者，让读者变成付费者。

## 技能定义

| 维度 | 说明 |
|------|------|
| **输入** | 内容、用户画像、平台规则 |
| **输出** | ConversionPlan（转化方案） |
| **依赖** | SKILL-401 结构审查 |
| **自动化** | 人工★★★☆☆ Agent★★★★☆ |
| **训练价值** | 极高（内容→传播） |

## 执行步骤

### Step 1：标题优化（10min）

标题 = 第一转化点：
- 公式：身份 + 情绪 + 悬念
- 好标题："29岁，她删掉了辞职信"
- 坏标题："一个女人的故事"
- A/B 测试：准备 3 个标题，选数据最好的

### Step 2：免费章节设计（10min）

免费章节 = 钩子区：
- 前 3 章 = 完整小故事（有起承转合）
- 第 3 章末 = 最大悬念
- 第 4 章开始 = 付费墙

> 关键：免费内容要让读者"上瘾"而不是"了解"

### Step 3：转化点设计（10min）

| 转化点 | 方法 | 指标 |
|--------|------|------|
| 曝光→点击 | 标题+封面 | 点击率 > 5% |
| 点击→追更 | 前3章质量 | 追更率 > 30% |
| 追更→付费 | 付费墙位置 | 付费率 > 5% |
| 付费→续费 | 内容稳定性 | 续费率 > 60% |

## 产出

1. ConversionPlan（含标题、免费区、付费墙设计）
2. A/B 测试方案
3. 转化漏斗模型

## 关键指标

- 各环节转化率
- 付费率
- 首章打开率

## 反模式（避免）

- ❌ 免费内容太水（读者不会为垃圾付费）
- ❌ 付费墙太早（还没上瘾就收费）
- ❌ 标题党（转化了但流失快）
- ❌ 只看阅读量不看转化率

## 资产沉淀

- 转化数据 → 优化付费墙位置和标题策略
