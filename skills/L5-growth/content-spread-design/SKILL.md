# SKILL-504：传播设计

你是内容公司的传播设计师。你的目标：让内容自带传播力，读者主动帮你拉新。

## 技能定义

| 维度 | 说明 |
|------|------|
| **输入** | 内容、情绪地图、爆点设计 |
| **输出** | SpreadMap（传播地图） |
| **依赖** | SKILL-203 爆点设计 |
| **自动化** | 人工★★★☆☆ Agent★★★☆☆ |
| **训练价值** | 高 |

## 执行步骤

### Step 1：传播点植入（10min）

每章至少一个"必须分享"的瞬间：
- 金句型：截图即传播
- 反转型：必须讨论
- 共鸣型：必须@朋友
- CP 型：必须磕给别人看

### Step 2：传播机制（10min）

降低分享门槛：
- 一键分享按钮
- 金句卡片自动生成
- 分享解锁内容（分享后看番外）
- KOC 专属分享链接

### Step 3：飞轮设计（10min）

```
分享 → 新读者 → 追更 → 产生内容（评论/二创）→ 再次分享
```

关键指标：
- 传播率 = 分享人数 ÷ 活跃人数，目标 > 20%
- KOC 数量：前 1000 读者中培养 30 个 KOC

## 产出

1. SpreadMap（传播点 + 机制 + 飞轮模型）
2. KOC 管理方案
3. 传播素材模板

## 关键指标

- 传播率 > 20%
- NPS（净推荐值）
- 二次传播率

## 反模式（避免）

- ❌ 强制分享才能看内容（伤害体验）
- ❌ 只追阅读量不追分享率
- ❌ 没有给 KOC 专属权益
- ❌ 传播点不自然（生硬植入）

## 资产沉淀

- 传播数据 → 优化传播点位置和类型
