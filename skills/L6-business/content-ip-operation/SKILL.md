# SKILL-604：IP运营

你是内容公司的IP运营。你的目标：把优质内容变成可授权、可衍生的IP资产。

## 技能定义

| 维度 | 说明 |
|------|------|
| **输入** | 成熟内容、读者数据、市场机会 |
| **输出** | IPBook（IP资产手册） |
| **依赖** | SKILL-603 内容产品化 |
| **自动化** | 人工★★★★★ Agent★☆☆☆☆ |
| **训练价值** | 低 |

## 执行步骤

### Step 1：IP资产盘点（15min）

梳理已有IP资产：
- 角色IP：核心角色的独立商业价值
- 世界观IP：可扩展的设定体系
- 故事IP：可改编的完整故事
- 金句IP：可传播的标志性台词

### Step 2：授权路径（15min）

| 授权类型 | 收入模式 | 时机 |
|----------|----------|------|
| 影视改编 | 买断+分成 | 有10万+读者后 |
| 有声改编 | 分成 | 有5万+读者后 |
| 游戏改编 | 买断+分成 | 有成熟世界观后 |
| 周边授权 | 授权费 | 有强角色IP后 |
| 海外授权 | 买断 | 有国内验证后 |

### Step 3：IP保护（15min）

- 版权登记
- 商标注册（角色名/作品名）
- 合同模板（授权范围/期限/分成）
- 侵权监控

## 产出

1. IPBook（IP资产清单+授权策略+保护方案）
2. 授权合同模板
3. IP价值评估

## 关键指标

- IP授权收入占比
- 单IP授权数量
- 衍生内容成功率

## 反模式（避免）

- ❌ 还没验证就谈IP（先有读者再谈授权）
- ❌ 授权不签合同（口头=没有）
- ❌ 独家授权给一家（除非条件极好）
- ❌ 不保护版权（被盗版后再维权成本高）

## 资产沉淀

- IPBook → 公司最核心的资产目录
