# 网文工厂 Profile

> 专为连载网文场景配置的内容工厂垂直包。

## 适用场景

- 起点中文网、番茄小说、晋江文学城等平台的连载网文
- 日更/周更模式，50+ 章长篇
- 以 IP 孵化 + 付费追更为商业目标

## 特化配置

### L0 主编 — 特化

| 维度 | 网文特化 |
|------|----------|
| 画像重点 | 平台读者画像（年龄/性别/阅读场景/付费意愿） |
| 情绪重点 | 核心情绪驱动（逆袭/甜宠/悬疑/热血） |
| 选题重点 | 平台热榜分析 + 竞品对比 + 品类缺口 |

### L1 设定师 — 特化

| 维度 | 网文特化 |
|------|----------|
| 世界观 | 需支撑 50+ 章长线展开 |
| 人物 | 主角必须让目标读者投射自己 |
| 关系 | 至少 3 组对立关系 + 1 条暗线 |
| 风格 | 口语化、节奏快、章末钩子 |

### L3 写手 — 特化

| 维度 | 网文特化 |
|------|----------|
| 章节长度 | 3000-5000 字 |
| 开头钩子 | 前 200 字必须钩住读者 |
| 更新节奏 | 日更 3000 字 / 周更 2 章 |

### L4 审稿 — 特化

| 维度 | 网文特化 |
|------|----------|
| 审查重点 | 一致性是底线、情绪强度达标、章末钩子到位 |
| 发布标准 | P0/P1 问题清零 |

### L6 商务 — 特化

| 维度 | 网文特化 |
|------|----------|
| 付费墙 | 免费比例 30-50%，卡在情绪高点 |
| 会员 | 首月优惠 + 提前看 + 专属番外 |
| IP 运营 | 角色经济 > 故事经济 |

## 推荐技术栈

| 层级 | 推荐选型 |
|------|----------|
| 发布平台 | 起点/番茄/晋江 |
| AI 辅助 | Claude 辅助生成 + 人工审校 |
| 数据采集 | 平台后台 + 评论区 |
| 社群 | 微信群/QQ群 |
| 更新频率 | 日更 3000 字 |
